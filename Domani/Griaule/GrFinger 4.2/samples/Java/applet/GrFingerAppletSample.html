GrFinger Applet Sample

This Applet requires a Java Virtual Machine to run. If you don't have it installed on your computer, please, visit Sun Developer Network and get Java Plugin for your browser.

Differently from the other samples, this one stores all enrolled templates in memory instead of a database. If you restart the applet, all enrolled templates will be lost.

You need a GrFinger supported fingerprint sensor or a BMP or JPEG fingerprint image to use this sample.

Visit us at www.griaule.com to learn more about GrFinger.
