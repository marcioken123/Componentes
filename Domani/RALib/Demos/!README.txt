Demos
-----

This folder contains some examples for demonstrating 
main possibilities of the R&A library.

We recommend to familiarize with examples in the following order:
  - RAControls, RADBTreeView, RADBMove, RAhtHints;
  - RAEditor;
  - RAI2;
  - RAI2_EndUser;
  - RANotepad;
  - RAdOctober.

Some demos contains !Readme.txt files.
Please read it before procede with demos.
