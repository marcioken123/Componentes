Installation:

Freeware Version
----------------------------

Delphi3
------------
                   Duck Barcode Component
                   ------------------------------------------
                   1. Goto Delphi menu Component -> Install Packages..
                   2. Add.. Button
                   3. Select Directory <Install Directory>\Lib\Delphi3
                   4. Open file Barcode30.dpl
                   5. OK Button
                   6. Goto Delphi menu Tools -> Environment Options
                   7. Select Library
                   8. Append value in Library Path: is <Install Directory>\Lib\Delphi3

                   Duck Barcode Component for Database
                   ---------------------------------------------------------------------
                   1. Goto Delphi menu File\Open
                   2. Select Directory <Install Directory>\Lib\Delphi3
                   3. Open file DBDB40.dpk
                   4. Push button Compile.
                   5. Push button Install.

                   Duck Barcode Component for Quick Report
                   ---------------------------------------------------------------------
                   1. Goto Delphi menu File\Open
                   2. Select Directory <Install Directory>\Lib\Delphi3
                   3. Open file DBCQ30.dpk
                   4. Push button Compile.
                   5. Push button Install.
