Send any problem reports report via email to mail@lmdsupport.com or use extended options at http://www.lmdsupport.com.  If you report problems or bugs please consider the following points:

  • Please mention your Delphi/BCB AND your LMD-Tools version in any case.
  • Please describe all steps to reproduce the problem.
  • If you use more than one IDE: Is the problem reproducable?
  • Which OS are you using? (Win2K, WinXP, Windows Vista, Windows 7, Windows 8, Server Releases etc.)

If necessary information parts are missing, it can be very time-intensive to reproduce your problem (especially when problems occur only under certain conditions).
