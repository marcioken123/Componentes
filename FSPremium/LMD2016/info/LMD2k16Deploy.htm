You may redistribute following LMD-Tools runtime packages royalty free for the purpose of executing applications created with LMD Innovative products (assuming you registered the corresponding product). You may not modify these packages in any way. Note: Under no circumstances any of the design time packages may be redistributed!

XX in package names represent the supported VCL version. Value range starts from 60 (Delphi 6). Following values are possible:

60, 70, 90: Delphi 6, Delphi 7 and 2005
100: BDS 2006 (Delphi / C++ personality), Turbo Versions
110: CodeGear RAD Studio 2007 (Delphi / C++ personality)
120: CodeGear RAD Studio 2009 (Delphi / C++ personality)
140: Embarcadero RAD Studio 2010 (Delphi / C++ personality)
150: Embarcadero RAD Studio XE (Delphi / C++ personality)
160: Embarcadero RAD Studio XE2 (Delphi / C++ personality)
170: Embarcadero RAD Studio XE3 (Delphi / C++ personality)
