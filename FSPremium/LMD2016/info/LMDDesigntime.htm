 Designtime Enhancements
All LMD VCL products share the same LMD Designtime Enhancements package which is installed automatically (dcllmdrtl11_XX.bpl or LMD.Vcl.Design.Rtl.dll, compiled versions only).

This package provides common designtime features for all LMD packages. Settings are possible via the About dialog of any LMD control (double click About property in object inspector). Following tabs are available:

 1. General
 Version Info about the LMD VCL package. In registered/full versions the RegUtility button is shown which starts your personal registration utility.

2. Support
Provides support options like email addresses, link to BTS (Bug Tracking System), newsgroups etc....

3. Settings
Use this page to enable/disable advanced general designtime editors and visual features for object inspector. Available options are:

  • Enabke/disable common property editors. TBrush, TCaption, THint, TShortCut and String editor can be switched on/off here.
  • Enable/disable visual object inspector enhancements. Visual enhancements for Font, Boolean and sets can be switched on/off here.

LMD-Tools Core specific options are:

  • Specify whether available labeled controls should be installed in IDE or not.
  • Specify whether designtime only LMD controls should be installed. These controls are used only for property/component editors and are only required, if you want to edit/modify certain editors

Note: Delphi/CBuilder must be restarted before changes have any effect.
