Introduction
Welcome to LMD DialogPack! LMD Dialog Pack consists of a number of components that not only emulate the Windows Vista TaskDialog and TaskDialogIndirect API functions on other operating systems but also provices enhancements over those APIs (e.g. custom header settings, extented text formatting, integration of checkboxes, dialog sheet controls etc).
There are components for enhancing the dialogs with edit controls and also replacing the Radio Button functionality with Check Boxes. Also included are three Password specific dialogs, that allow you to capture password information by dropping a component on your form and writing a line or two of validation code.


 First steps
If you encounter any problems during installation review the Installation notes (WIN32) or (VCL.NET). C++ user of BDS 2006 or later should read the corresponding special notes.
The documentation contains step by step explanations, start with Dialog Dissection

Location of demos
 Demo projects can be found in the \lmddialog folder in \demos below the installation directory. Please note that not all projects can be used in earlier Delphi releases because features of newer IDE releases were used (like e.g. namespaces etc.).
